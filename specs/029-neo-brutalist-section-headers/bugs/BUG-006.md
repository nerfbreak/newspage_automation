# Bug Report: Unstyled Header and Divider in Promotion Comparison

**Type**: UI/UX Inconsistency (Design System Violation)
**Severity**: Medium
**Feature**: 029-neo-brutalist-section-headers
**Reported**: 2026-07-10

## Description
During the Spec 029 codebase sweep, an unstyled HTML header (`<h3>Smart Comparison Engine</h3>`) and a standard markdown divider (`st.markdown("---")`) were discovered in `pages/3_promotion_comparison.py`. These elements violate the strict Neo-Brutalism design system.

## Reproduction Steps
1. Navigate to "Promotion Comparison" module.
2. Upload a valid ZIP file to trigger the comparison view.
3. Observe the "Smart Comparison Engine" header and the horizontal line above it.
4. **Expected**: Boxed Neo-Brutalist header with a thick 3px solid black divider.
5. **Actual**: Plain unstyled `<h3>` text and a thin grey 1px divider.

## Root Cause
The page used legacy inline HTML and standard markdown dividers instead of the approved `.section-header-underline` CSS class and custom thick dividers.

## Resolution Plan
- Replace `st.markdown("---")` with `<div style='border-top: 3px solid #0F172A; margin: 24px 0px;'></div>`.
- Replace `<h3>...</h3>` with `<div class='header-wrapper-left'><span class='section-header-underline'>...</span></div>`.
- This ensures 100% compliance across the entire project.

**Status**: Patched (Commit `3ee61bd`)
