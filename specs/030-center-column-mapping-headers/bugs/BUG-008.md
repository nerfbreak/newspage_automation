# Bug Report: Column Mapping Header Positioned Too Low (BUG-008)

**Type**: Visual alignment
**Severity**: Low
**Feature**: 030-center-column-mapping-headers
**Reported**: 2026-07-10
**Status**: Patched

## Description

The "Column Mapping" section headers added by Spec 030 use `header-wrapper-center` which applies `margin-top: 32px`. This pushes the header too far down from the content above (file upload box / data preview), making it appear misaligned. User wants the text centered within its Neo-Brutalist box without excessive top spacing.

## Root Cause

`header-wrapper-center` has `margin-top: 32px !important` — designed for section headers that follow larger content blocks. For the Column Mapping headers, which sit between an upload section and dropdown columns, the spacing is excessive.

## Fix

Replace `header-wrapper-center` with `header-wrapper-center-notop` (existing class, `margin-top: 4px`) on both instances:
- `pages/4_stock_mutation.py:116`
- `pages/1_inventory_adjustment.py:309`
